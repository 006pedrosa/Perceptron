ALGORITMO DO PERCEPTRON

Autor: Pedro Guimarães Rosa

Tutorial

Entrada: a entrada deve seguir um modelo de matriz, sendo possível no máximo 2 valores por linha
Exemplo:
1 0
0 1 0
0 0 1 0
O sistema suporta qualquer quantidade de entradas por linha, e várias linhas por teste. Para executar uma modificação do tipo de método a avaliar, deve-se aplicar um novo teste.
Para os parametros de entrada temos:
Aprendizado: taxa de aprendizado com que os pesos serão reajustados.
Bias: valor do Bias.
Tempo: número máximo de ciclos para executar o calculo.
SELECIONE UM MÉTODO: Escolha entra AND, OR ou XOR.
