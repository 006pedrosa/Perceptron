ALGORITMO DO PERCEPTRON

Autor: Pedro Guimarães Rosa

Tutorial

Entrada: a entrada deve seguir um modelo de matriz, sendo que a ultima coluna é reservada para o resultado esperado
Exemplo:
1 0 0
1 1 1
0 0 0
Neste caso, temos que a matriz representa uma tabela verdade AND, onde o último digito em vermelho representa a solução da operação para aquelas entradas.
Para os parametros de entrada temos:
Arestas: número de arestas do perceptron.
Aprendizado: taxa de aprendizado com que os pesos serão reajustados.
Bias: valor do Bias.
Tempo: número máximo de ciclos para executar o calculo.
